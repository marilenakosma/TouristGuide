# Travel App

A Java Swing-based desktop application that allows users to plan transportation routes between major Romanian cities. The app features an interactive map (powered by JXMapViewer) and a user-friendly interface for selecting destinations and modes of transport.

---

## 🚀 Features

- Interactive map with clickable locations (e.g., airport, museum)
- Route visualization with markers and lines
- Combobox-based trip planner for selecting:
  - Departure and arrival cities
  - Mode of transportation: Bus, Train, or Plane
- Visual feedback with buttons and images
- Predefined city-to-city routes

---

## 🧭 Technologies Used

- **Java Swing** – UI components
- **JXMapViewer2** – Map rendering and interaction
- **Maven** (optional) – Dependency management
- **Java 8+**
